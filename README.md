IEB Menu is a next-generation mod menu for Geometry Dash, featuring over 100 hacks, customizable labels, a replay bot, and more. It is built to be fast, reliable, and visually stunning.

## How to use

1. On PC, press **TAB** to open and close the mod menu.
2. Toggle your desired hacks and enjoy the game!

## Features

- **100+ hacks**: Speedhack, Noclip, Bypass, and more.
- **StartPos Switcher**: Smart StartPos functionality for easier level practice.
- **Fully Customizable Labels**: Add and theme labels to your heart's content.
- **Internal Recorder**: Capture your gameplay with high quality.
- **Replay Bot**: Record and play back your runs.
- **Theme Customizability**: Fully personalize the menu's look.
- **Cross-platform**: Supports Windows, macOS, Android, and iOS.

## Build Instructions

1. Ensure you have the [Geode SDK](https://geode-sdk.org) installed.
2. Run the following command in the project directory:
   ```bash
   geode build
   ```
3. Copy the resulting `.geode` file to your Geometry Dash `mods` folder.
