# IEB Menu

A **premium** mod menu for Geometry Dash, developed by **IEB Studio**.

## How to use

1. On PC, press `TAB` to open and close the mod menu. For mobile, use the floating button to toggle the menu!
2. Enable any hack or customization you desire.
3. For support and scripts, visit the **AI Scripts** tab or join our community.

## Features

- **Chroma (RGB)**: Smoothly cycle through colors for a premium look.
- **AI Scripts**: Generate and manage scripts using natural language prompts.
- **Watermark**: Translucent branding for a professional aesthetic.
- 100+ legacy hacks included.
- Responsive design for all platforms.
